## Product Philosophy

本产品是一款深度融合**第二语言习得（SLA）**理论与**前沿生成式AI（Generative AI）**的跨学科英语精听与认知训练系统。与市面上碎片化的背单词或被动听力软件不同，本系统秉持“**主动产出推动输入内化**”的核心理念，为学习者构建了一个高动态、沉浸式的语言交际场景。

系统的核心训练管线分为三大递进维度：

1. **选择性注意与整体感知 (Selective Attention & Holistic Perception)**
   系统隐藏全文，要求学习者完全依赖听觉信号捕获宏观信息。这有助于打破日常阅读中对视觉符号的依赖，重建大脑的语音环路（Phonological Loop）。
2. **多模态认知加工 (Multimodal Cognitive Processing)**
   * **Write a One-Sentence Summary**: 训练长时记忆中的概念整合能力。
   * **Main Ideas Summary**: 考察核心论点捕捉（Coverage）与文章宏观逻辑结构（Structural Accuracy）的还原度，杜绝传统英语教学中“只听孤立词汇”的盲区。
3. **高动态精细化查词与语境整合 (Dynamic Lexical Search & Contextualization)**
   系统通过深度对接大语言模型，允许用户自由圈选陌生表达，不仅提供精准定义，更垂直扩展出**派生词（Derivatives）**与**常见搭配（Collocations）**，实现词汇网络在学术/日常语境下的深度蔓延。
